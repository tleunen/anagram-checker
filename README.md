# Anagram-Checker

Simple module to check if multiple string are anagrams.

## Example

```
var AnagramChecker = require('anagram-checker');

AnagramChecker('silenced', 'licensed', 'Declines'); // true
```

## Install

Install with npm:

`npm install anagram-checker`
